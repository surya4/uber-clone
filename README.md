## Basic Uber Clone using Node, Kafka, GoogleMap and Cassandra

### Configuration
- **Platform:** node
- **Framework**: express
- **Template Engine**: ejs
- **CSS Framework**: bootstrap
- **CSS Preprocessor**: css
- **Build Tool**: none
- **Unit Testing**: mocha
- **Database**: cassandra
- **ORM**: 
- **Deployment**: aws

### How to build: 
docker build -t uber-clone .

### How to run 
docker run --rm -it -p 3000:3000 uber-clone
